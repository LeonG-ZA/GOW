G.O.W

G.O.W is a community driven Ultima Online server emulator written in c#.

Version

4.4

Installation

Getting started with G.O.W is quite easy.

Windows

Just run  the exe

OSX

 brew install mono 
 make 

Ubuntu

 apt-get install mono-complete 
 make
